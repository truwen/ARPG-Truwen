# ARPG-Truwen

First playable top-down ARPG prototype using the Canvas API.

## Controls
- `W`, `A`, `S`, `D`: move the player
- `Space` or left mouse click: melee attack

## Prototype Features
- Real-time game loop (`update` + `render`)
- Player and one chasing enemy
- Basic melee hit detection and damage
- Health for player and enemy
- Simple placeholder rendering with circles/bars
